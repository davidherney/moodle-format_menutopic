COURSE FORMATS menutopic
============================

Package tested in: moodle 3.5+, 3.6+, 3.7+, 3.8+, 3.9+, 3.10+

QUICK INSTALL
==============
Download zip package, extract the menutopic folder and upload this folder into course/format/.
